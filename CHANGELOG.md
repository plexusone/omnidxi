## [v0.1.0] - 2026-04-27

### Highlights

- Initial release of omnidxi - batteries-included DXI client for Go
- MultiTracker for sending events to multiple providers simultaneously
- Bundled Amplitude and Mixpanel provider adapters

### Added

- Re-exports of all core types from omnidxi-core ([`f254df6`](https://github.com/plexusone/omnidxi/commit/f254df6))
- MultiTracker for multi-provider event dispatch with aggregated error handling ([`f254df6`](https://github.com/plexusone/omnidxi/commit/f254df6))
- Thread-safe Add method for runtime provider addition ([`f254df6`](https://github.com/plexusone/omnidxi/commit/f254df6))
